AI-powered mobile app that estimates calories from food images.

Features:

Image upload → calorie estimation
Daily calorie tracking
Weight tracking + graphs
Clean mobile UI (Flutter)

Tech Stack:

Flutter
Supabase (auth + DB)
OpenAI API (image analysis)

Status:
MVP nearly complete, preparing for launch
